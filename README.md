# Filmorate
### Приложение для оценки просмотренных фильмов, с возможностью регистрации новых пользователей, добавления в друзья и ставить лайки понравившимся фильмам.

---
## Основные возможности
- Добавление режиссеров к фильмам
- Список самых популярных фильмов по жанрам и годам выпуска
- Функция "Поиск"
- Функция "Общие фильмы"
- Функция "Рекомендации"
- Удаление фильмов и пользователей
- Функция "Отзывы"
- Функция "Лента событий"

## Используемые технологии
- Spring Boot
- Spring Data (JDBC)
- H2
- JUnit
